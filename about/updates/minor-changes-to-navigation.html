Some minor cosmetic changes have been done to improve navigation throughout this site:

  • The navigation between pages has been changed from text links to graphic arrow links. Almost every page will now allow to navigate to the previous or next page (if available) within the same hierarchy, and it will allow to navigate to its parent page, unless if it is the top page.
  • The titles of pages about dated topics, such as the main pages about kings, have been modified to include a start and end date of the period covered by the page.


The pages about the known kings of the first four dynasties have already been migrated to the new layout. This includes information about their funerary monuments, if such monuments have been identified.
